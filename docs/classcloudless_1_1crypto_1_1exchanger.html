template<typename Hash = SHA256>
class cloudless::crypto::exchanger< Hash >

A class to help exchange a shared secret securely using FHMQV algorithm.

Template Parameters:
Hasha hashing algorithm to be used with FHMQV. Defaults to SHA256.
