virtual void cloudless::detail::thread::body ( ) [protected, pure virtual]

A pure virtual function to be overrided by a child of this class.

This function represents the body of instructions and expressions to be ran inside the thread.

Note: This function will be called within an indefinite while-loop, if you wish to conclude the operation of this thread, you must call stop() from within this function.
